# Vue Boilerplate Dashbaord

## Environments

```
1. Production

==> Branch Name: main

2. Development

==> Branch Name: develop

```

## Install

```sh

git clone https://github.com/awal-solution/admin-dashboard-element-plus.git
cd  admin-dashboard-element-plus.git
yarn
```

## Start development

```sh
yarn dev
```

## Build for production

```sh
yarn build
```
